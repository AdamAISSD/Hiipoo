## 4. Principales puntos de venta

### 4.1 Diseñada para cámaras Sony CFexpress Type A

No es una tarjeta SD. Es una tarjeta **CFexpress Type A** para cámaras y videocámaras compatibles, especialmente modelos Sony con ranura Type A.

### 4.2 Velocidad de hasta 800MB/s

Pertenece a la clase de velocidad 800MB/s, lo que permite manejar archivos fotográficos grandes, ráfagas rápidas y flujos de trabajo de video 4K con más fluidez que una tarjeta SD común.

### 4.3 Adecuada para video 4K

Recomendada para formatos como **XAVC HS 4K, XAVC S 4K y XAVC S HD**, según el modelo de cámara y la configuración. Para XAVC S-I, 8K, All-Intra o modos de alto bitrate, el comprador debe confirmar compatibilidad antes de comprar.

### 4.4 Ráfagas de alta velocidad

Ideal para fotografía deportiva, vida silvestre, viajes, bodas, eventos y situaciones donde cada momento cuenta.

### 4.5 Protección IP68 contra agua

La protección IP68 es útil para fotografía al aire libre, viajes, lluvia y entornos exigentes. Para publicaciones profesionales, conviene acompañarla con reportes de prueba o documentación oficial si están disponibles.

### 4.6 Resistencia a rayos X para viajes

Adecuada para controles de seguridad en aeropuertos y viajes internacionales. Evite afirmar que la tarjeta es inmune a todo tipo de radiación.

### 4.7 Resistencia a temperaturas amplias

Temperatura de uso: aproximadamente **-15°C a 75°C**.  
Temperatura de almacenamiento: aproximadamente **-40°C a 85°C**.  
Adecuada para fotografía exterior en ambientes fríos o calurosos.

### 4.8 Reemplazo por 3 años

El servicio de reemplazo por 3 años ayuda a reducir la preocupación del cliente. El vendedor debe aclarar condiciones de garantía, daños no humanos, comprobante de compra y si la recuperación de datos está incluida.

## 5. Recomendación de capacidad y versión

- **256GB / 512GB:** Fotografía, viajes, video 4K diario y uso general de creadores.
- **512GB / 1TB:** Videos cortos, bodas, trabajos comerciales y usuarios que necesitan más espacio.
- **Versiones P-SLC 80GB / 160GB / 320GB:** Mejor opción para grabación larga de alto bitrate y escritura sostenida más estable.
- **Serie profesional Petpet 4.0:** Para compradores que necesitan mayor velocidad, flujo de video profesional o rendimiento PCIe 4.0.

Recordatorio para vendedores: **“Hasta 800MB/s” significa rendimiento pico, no escritura sostenida garantizada todo el tiempo.**
